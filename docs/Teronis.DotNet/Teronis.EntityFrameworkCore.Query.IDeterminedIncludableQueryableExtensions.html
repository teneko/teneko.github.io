    Include<EntityType, PreviousPropertyType, PropertyType>(IDeterminedIncludableQueryable<EntityType, PreviousPropertyType>, Expression<Func<EntityType, PropertyType>>)

    Specifies related entities to include in the query results. The navigation property to be included is specified starting with the type of entity being queried (EntityType). If you wish to include additional types based on the navigation properties of the type being included, then chain a call to ThenIncludeIf<EntityType, PreviousPropertyType, PropertyType>(IDeterminedIncludableQueryable<EntityType, IEnumerable<PreviousPropertyType>>, Expression<Func<PreviousPropertyType, PropertyType>>, Boolean) after this call.
