    Class CollectionChangeHandler<TItem>.ItemReplacableCollectionChangeBehaviour

    Implements the behaviour to replace an item at a given index on request. It is the contrary implementation of CollectionChangeHandler<TItem>.DefaultCollectionChangeBehaviour as it would not replace existing items.
