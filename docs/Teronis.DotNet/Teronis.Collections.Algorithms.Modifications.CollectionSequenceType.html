    Enum CollectionSequenceType

    Namespace: Teronis.Collections.Algorithms.Modifications
    Assembly: Teronis.NetStandard.Collections.Algorithms.dll
    Syntax
    public enum CollectionSequenceType : int

    Fields

    Name Description
    Ascending

    The items are sorted ascended, e.g. index 0 is smaller than index 1 and therefore comparable.
    Descending

    The items are sorted descended, e.g. index 0 is greater than index 1 and therefore comparable.
    Sequential

    The items are in any order and therefore not comparable but equality comparable.
