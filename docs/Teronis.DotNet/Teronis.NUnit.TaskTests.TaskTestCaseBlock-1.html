    Class TaskTestCaseBlock<TDerived>

    This class is inteded to be inherited. In its derived class it is intended to implement a static member of such derived class. There are three ways to initialize the static member.

    The manual way is only achievable with TaskTestCaseBlock. See coments there.

    The more automatic way is to use TaskTestCaseBlock<TDerived>. It passes a TaskTestCaseList to its base type TaskTestCaseBlock<TDerived>. Because you can't to call AddTest(TaskTestCaseList, Func<CancellationToken, Task>, Nullable<String>) but AddTest(Func<CancellationToken, Task>, Nullable<String>), you have to Initialize() after the creation of derived class.

    Last but not least you can you just skip the instantiation of the static member. You then can leave the instantiaton to AssemblyTaskTestCaseBlockStaticMemberCollector in conjunction with TaskTestCaseBlockStaticMemberProviderAttribute and TaskTestCaseBlockMemberAssigner.
