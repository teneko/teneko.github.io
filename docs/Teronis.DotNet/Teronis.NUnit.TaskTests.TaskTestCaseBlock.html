    Class TaskTestCaseBlock

    This class is inteded to be inherited. In its derived class it is intended to implement a static member of such derived class. In contrast to TaskTestCaseBlock<TDerived> this class has only the manual way to initialize the static member. You should store a static instance of TaskTestCaseList in derived class and pass it always as parameter to AddTest(TaskTestCaseList, Func<CancellationToken, Task>, Nullable<String>). By calling GetTestCases() you get the assertable tasks, but before that you need prepare them by calling PrepareTasksAssertion(CancellationToken).
