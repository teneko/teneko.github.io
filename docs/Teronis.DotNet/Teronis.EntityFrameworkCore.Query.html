    CollectionConstantPredicateBuilder<SourceType, ComparisonItemType>

    The collection constant predicate builder produces one predicate that includes the comparison between the source of type SourceType and each collection item of type ComparisonItemType. Each collection item can be itself a source of type ComparisonItemType and therefore be compared to collection item of a collection that is member of ComparisonItemType and so on.

    (e.g. src1.A == cmpItem1.Z
