    Class OrderedDictionary<K, V>

    A dictionary object that allows rapid hash lookups using keys, but also maintains the key insertion order so that values can be retrieved by key index.

    Remarks

    Similar to the way a DataColumn is indexed by column position and by column name, this advanced dictionary construct allows for a very natural and robust handling of indexed structured data.
